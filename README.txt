Pour lancer l'exécution :

1- Télécharger et dezipper l'archive
2- Dans le terminal, lancer Julia (avec la commande "Julia")
3- Se déplacer jusqu'au dossier racine (en appuyant sur ";" pour entrer en mode shell, 
   puis se déplacer jusqu'au dossier avec la commande "cd"). 
4- Une fois au bon emplacement, exécuter le main, avec la commande "include("mainMinBP.jl")

N.B : Dans le cas d'une génération d'instances aléatoire, il faut modifier le chemin d'accès 
      (ligne 62 de mainMinBP.jl, dans la variable target) pour qu'il colle au chemin de la 
      machine sur laquelle on travaille.
